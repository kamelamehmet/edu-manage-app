# src

Project `src` structure created for the edu-manage-app frontend.

Folders:
- api — pocketbase client wrapper and services
- auth — AuthProvider and hooks
- components — shared UI components
- features — feature modules
  - courses
  - grades
  - payments
  - schedule
- pages — top-level pages
- routes — routing setup
- store — state management
- hooks — shared hooks

These files are placeholders. Implement module contents as needed.
